# Homebrew Tap de fan2dev

Tap de [Homebrew](https://brew.sh) para las aplicaciones de fan2dev.

## GRUJ Cleaner

Limpiador profundo de cachés de entornos de desarrollo en macOS (Xcode, Android/Gradle,
Flutter, gestores JS, Docker y más), sin tocar tus datos.

```bash
brew install --cask fan2dev/tap/gruj
```

- Web: <https://gruj.alejandrofan2.dev>
- Las actualizaciones las gestiona la propia app (Sparkle); por eso el cask lleva `auto_updates true`.
